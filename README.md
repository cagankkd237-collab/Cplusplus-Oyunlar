# Cplusplus-Oyunlar
C++ dilini öğrenirken geliştirdiğim konsol tabanlı oyun projelerim.

## İçerideki Oyunlar
### 1. XOX Oyunu
* **Dil:** C++
* **Özellikler:** İki kişilik oynanabilir. Oyuncular sırasıyla 1 ile 9 arasında bir sayı girerek hamlelerini yaparlar.

## 2. RPG Dovus Oyunu
* **Dil:** C++
* **Özellikler:** Oyuncu 1 ve ya 2 sayısını girerek hamlelerini yapar.Rastgele hasar verme ve ya can alma mekanikleri mevcuttur. Oyuncu 3 sayısını girerek oyundan çıkış yapabilir.

## Nasıl :Çalıştırılır?
Herhangi bir C++ derleyicisinde (Visual Studio Code, Dev-C++ vb.) '.cpp' dosyalarını açıp derleyerek direkt konsol üzerinden oynayabilirsiniz.
